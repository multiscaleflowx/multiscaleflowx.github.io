Software Overview

We develop and use the following software:

  • Molecular dynamics (MD) (e.g. LAMMPS, mdFOAM)
  • Enskog-Vlasov solvers (in-house codes)
  • Event driven molecular dynamics (EDMD)
  • Discrete Element Method (DEM) (e.g. LIGGGHTS, LAMMPS)
  • Peridynamics (PD) (e.g. LAMMPS)
  • Lattice Boltzmann Method (LBM) (in-house codes)
  • Direct Simulation Monte Carlo (DSMC) (e.g. dsmcFOAM, SPARTA)
  • Discrete Velocity Method (DVM) (in-house coddes)
  • Computational Fluid Dynamics (CFD) (e.g. OpenFOAM)
  • Coupling software (e.g. MUI)
