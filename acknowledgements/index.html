Website Design and Implementation

This website is built using Jekyll, inspired by the elegant designs of D. Allan Drummond and Woo Young Ahn (these were both in turn inspired by Trevor Bedford's site). Rather than branching any of these templates, we started from a basic Bootstrap template, and adopted/adapted bits of other code during customization.

This site’s source code is freely available on GitHub and deployed using GitHub Pages.
